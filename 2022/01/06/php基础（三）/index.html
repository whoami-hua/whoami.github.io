二：PHP文件处理

1.表单处理过程

通过句柄来确定从哪里开始读取文件

<?php
$name = "1.txt";					//定义文件名字
$file = fopen($name,'r')			//用读的权限打开文件，并获取句柄
echo fread($file,filesize($name));		//读取整个文件
fclose($file);						//关闭文件
?>
2..fopen()函数
$file = fopen($name,'r')			//用读的权限打开文件，并获取句柄

3.fread()函数

fread($file,filesize($name));		//读取整个文件
4.fgets()函数

读取文本文件中的一行

<?php
$file = fopen('1.txt','r');
while(!feof($file)){				//判断指针是否已经到达文件末尾，没有到文件末尾就循环读取文件
echo fgets($file)."<hr>";			//读取一行文本
}
fclose($file);
?>
5.fgetss() 函数

读取文件中的一行字符，并去掉所有HTML和PHP标记

<?php
$file = fopen('1.txt','r');
while(!feof($file)){				//判断指针是否已经到达文件末尾，没有到文件末尾就循环读取文件
echo fgetss($file)."<hr>";			//读取一行文本,并去掉所有HTML和PHP标记
}
fclose($file);
?>
6.fgetc() 函数

读取文件中的一个字符

<?php
$file = fopen('1.txt','r');
while(!feof($file)){				//判断指针是否已经到达文件末尾，没有到文件末尾就循环读取文件
echo fgetc($file)."<hr>";			//读取文件中的一个字符
}
fclose($file);
?>
7.file_get_contents() 函数

无须经过打开文件及关闭文件操作就可读取文件中的全部内容 ，

<?php
echo file_get_contents(1.txt);			//读取文件的全部内容
?>

8.file_put_contents()函数

以追加的方式将内容写进文件中

<?php
$file = 'log.txt';			 //要写入文件的文件名，如果文件不存在，将会创建一个log.txt
$content = "helleworld";
if(file_put_contents($file, $content,FILE_APPEND))
{
echo "success<br />";
}
echo file_get_contents($file);
?>
9.fwrite() 函数

以将一个字符串写入到文本文件中

<?php
$file = 'log.txt’;
$handle = fopen($file, 'w') 
$data = 'This is the data';
fwrite($handle, $data);
fclose($handle);
?>

10.file()函数

把文件读入到一个数组中

<?php
print_r(file("test.txt"));
?>

输出

Array
(
[0] => Hello World. Testing testing!
[1] => Another day, another line.
[2] => If the array picks up this line,
[3] => then is it a pickup line?
)

<?php
if(copy('test.txt','./data/bak.txt')) //复制文件
echo '文件复制成功';
else echo '文件复制失败';
unlink('./test.txt'); //删除文件
?>
