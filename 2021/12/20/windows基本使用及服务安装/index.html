一：常用DOS命令

1.DOS概念

2.ipconfig
命令 说明
ipconfig 查看ip地址
ipconfig /release 释放版本机现有的ip地址
ipconfig /renew 重新向dhcp获取ip地址
ipconfig /all 显示完整的ip地址
3.用户及用户名
命令 说明
net user 账号名 密码 /add 添加一个新用户
net user 账号名 /del 删除用户
net user 账号名 查看指定用户的详情信息
net localgroup 产看本地用户组
4.netstat
命令 说明
netstat -a 查看本地机器开放端口
netstat -r 列出当前的路由信息
netstat -n 不对名字进行解析
5.cd命令
命令 说明
cd\ 返回根目录
cd .. 返回上一层目录
cd 进入指定目录
6.其他常用命令
命令 说明
dir 查看文件夹下的文件或目录
mkdir(md) 新建文件夹
rmdir(rd) 删除文件夹
copy 复制文件或文件夹
del 删除文件
ren 改名
type 查看文本文件
7.防火墙命令
netsh advfirewall firewall add rule name=icqtest dir=in action=allow protocol=tcp localport=3389     #增加本地的TCP的3389端口对外开放
