Liunx用户和权限

2021-12-19 CentoS 7基础 468 字
Linux

一：Liunx用户与用户组

1概念

2.创建用户
命令 说明
useradd 用户名 创建用户，同时创建与用户名同名的组
useradd –g 组名 用户名 创建用户并指定用户组
useradd 用户名 –s xxx 创建用户并指定用户登录后所使用的shell，如果值为 /sbin/nologin 或 者 /bin/false时，表示禁止该用户登录
usermod –g 用户组 用户名 修改用户所在用户组
usermod –l 新用户名 用户名 修改用户名
userdel 用户名 删除用户，不删除用户的家目录（-r 参数同时删除家目录）
3.创建用户组
命令 说明
groupadd 创建用户组
groupdel 删除指定用户组
4.用户管理
命令 说明
su 用户名 切换用户身份，如果不带参数，则切换为root身份
id 用户名 显示指定用户信息，包括用户编号、用户名，组编号、组名
whoami 显示当前登录用户名

二：Liunx权限控制

1.概念

2.文件夹/文件权限
命令 说明
chmod u+x test.txt 为文件的所有者附加执行权限
chmod u=rwx test.txt 为所有者赋予所有权限
chmod a=rwx test.txt 为所有人（所有者、所属组、其他人）赋予所有权限
chmod 755 test.txt 为文件赋予执行权限：rwxr-xr-x
常用组合：777最高权限，644普通文件权限，755执行权限
3.文件夹/文件所有者
命令 说明
chown icq test 将test文件的所有者修改为icq
chown icq:icqtest test 修改test文件的所有者和所有组修改为：icq、icqtest

4.文件夹/文件所属组
命令 说明
chgrp icqtest test.txt 将 test.txt 文件的所属组由 root 更改为 icqtest

5.显示、设置文件的默认权限
命令 说明
umask 022 改变基础权限
