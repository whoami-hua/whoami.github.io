Linux

一：liunx常用命令

1.关机命令
命令 说明
poweroff 立刻关机
halt 立刻关机（需要root权限）
shutdown -h now 立刻关机
init 0 关机
2.重启命令
命令 说明
init 6 重新启动
reboot 立刻重启
shutdown -r now 立刻重启
3.查看文件夹下的文件
命令 说明
ls 查看文件夹下的文件
ls -l 查看文件夹下的文件的详细信息
ls -lR 递归查看文件夹下的文件的详细信息
ls -a 查看隐藏文件
4.创建文件夹及删除文件夹
命令 说明
mkdir 新建文件夹
mkdir -p 递归创建文件夹
mkdir . 创建隐藏文件夹
rm 删除文件或文件夹
rm -rf 递归删除文件或文件夹
5.新建文件
命令 说明
touch 新建文件夹
touch -t 修改文件的时间
vim 新建并编辑文件
6.复制
命令 说明
cp 源文件 目标文件 复制文件
cp -r 递归复制
7.剪切移动重命名
命令 说明
mv 源文件 目标文件 移动文件（重命名）
8.查看
命令 说明
cat/grep/tail/head 查看
more 分页查看
9.辅助命令
命令 说明
Tcb 自动补齐
Ctrl+L 清屏
Ctrl+K 清空至行首
Ctrl+U 清空至行尾
Ctrl+C 中断执行命令
10.打包与解包
命令 说明
tar -cvf 打包
tar -xvf 解包
11.压缩命令
命令 说明
tar -zcvf 打包（gz)
tar -zxvf 解包（gz）
zip 打包（zip)
unzip 解包(zip)
12.防火墙命令
命令 说明
firewall-cmd –reload 防火墙重新加载配置
firewall-cmd –state 查看防火墙状态
firewall-cmd –zone=public –list-ports 查看防火墙开放那些端口
firewall-cmd –zone=public –list-service 查看接受那些连接附服务
firewall-cmd –zone=public –add-port=80/tcp –permanent 添加80端口为永久开放
firewall-cmd –zone=public –add-service=http –permanent 永久添加http连接服务
systemctl stop firewalld 关闭防火墙
systemctl start firewalld 开启防火墙
14.网络类命令
命令 说明
ip a 查看ip地址
ip route 查看路由信息
ss -antlp 查看开放那些端口
nslookup 域名解析为ip
traceroute 路由跟踪
netstat -antlp 查看运行的端口
15.服务类命令
命令 说明
ps -aux 查看进程
ps -A | grep 查看指定进程
top 查看cpu
kill -9 结束进程
w 查看登录信息
who 查看登录信息
16特殊操作
命令 说明
screen -S name1 创建新窗口
screen -ls 列出所有后台的会话
chattr +i 文件添加保护

vim编辑器的使用

1.vim工作模式

2.命令模式
命令 说明
i 光标所在处左侧输入
I 光标跳转行开头
o 光标所在的下一行处输入新的一行
O 光标所在的上一行处输入新的一行
a 光标所在处右侧输入
A 光标跳转行尾
wq 保存并退出
q 退出不保存
/$ 搜索
：n,ns/n/n/g(n为参数） 替换第几行里的字符
3.输入模式
命令 说明
o 移动到这一行的最前面字符处
$ 移动到这一行的最后面字符处
n 重复前一个搜寻的动作
N 重复前一个搜寻的动作
x 为向后剪切一个字符
dd 剪切游标所在的那一整行
ndd 多行剪切
u 一直撤销
p 粘贴
